用 Hexo + Github 建立靜態網頁

Posted on 2017-10-14 Edited on 2020-01-20 Views:
Words count: 1.9k Reading time ≈ 3 mins.

之前有概略介紹過靜態網頁跟 Github Pages（傳送門），這邊就不贅述了。沒有 Github 帳號的先申請一個（點我申請）。建立一個 repository，待會會用到，並將這個 repository 命名為 username.github.io（username 請填自己的，不要真的打 username）。

在安裝 Hexo 之前得先檢查電腦上是否已經安裝 Nodejs 跟 Git
