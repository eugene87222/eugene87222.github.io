國網 / 爬蟲 / PTT - 1

Posted on 2018-02-24 |

應徵上國網中心工讀生之後第一件事就是練習 python 爬蟲，爬 PTT、facebook、MOBILE01，這幾天正在做 PTT 的部分。

環境

  • Windows 10 家用版
  • python 3.6.4

安裝 library

  • requests (取得網頁原始碼)

  • BeautifulSoup4 (處理網頁原始碼)

爬蟲

取得網頁原始碼

先把要用到的 library import 進來

自訂一個函數 get_page_content 用來取得網頁原始碼，requests.get 裡面有很多參數，url 就是我們要爬的頁面的網址，這邊設定 cookies 的用意在於 PTT 有些版會要求年齡 18 歲以上才能進入，我們將 over18 設為 1 (已滿 18 歲) 並存入 cookies。

這邊以八卦版為例，八卦板首頁網址為 https://www.ptt.cc/bbs/Gossiping/index.html ，將網址傳入 get_page_content 中，requests.get 會回傳一個 Response 物件，透過 .text 可以取得網頁原始碼，
